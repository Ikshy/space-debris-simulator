# Space Debris Tracking Simulator

**A Python project to visualize satellites, simulate orbital debris collisions, and calculate orbital decay.**

---

## Features
- Load real satellite TLE data and compute positions over time
- Visualize orbits in 3D around Earth using Matplotlib
- Calculate collision probabilities between satellites
- Easy to update TLE file to track more satellites

---

## Tech Stack
- **Python 3.14+**
- **sgp4** (orbital mechanics)
- **NumPy & Pandas** (data processing)
- **Matplotlib** (3D visualization)
- **CSV module** (TLE parsing)
